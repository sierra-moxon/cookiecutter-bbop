Install the latest
===================

To install the latest version of {{cookiecutter.project_name}} simply run:

`pip3 install {{cookiecutter.project_name}}`

OR

`poetry add {{cookiecutter.project_name}}`

OR

`pipenv install {{cookiecutter.project_name}}`
